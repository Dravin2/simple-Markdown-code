####Ordered Lists

Ordered list are created using :"1." + Spance:  


1. Ordered list item  
2. Ordered list item  
3. Ordered list item  


#### Unordered List

Unordered list are created using : "*" + Space:   

*  Unordered list item  
*  Unordered list item  
*  Unordered list item  

 or using "-" + Space:  
 
 - Unordered list item  
 - Unordered list item  
 - Unordered list item
